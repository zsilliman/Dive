5 // This is class for decoding FLAC files with native encoding. It does not
6 // support Ogg files with FLAC data.
7 //
8 // FLAC supports up to 8 channels (7.1 stereo), though SDL is limited to
9 // 6 channels (5.1 stereo). Flac channel interleavings are compatible with
10 // SDL, so they are preserved.
